Less is More

A stripped-back design system emphasizing whitespace, clean typography, and restrained color for maximum clarity and focus.

Explore Components View Examples

Design Principles

W

Whitespace First

Use whitespace to separate concerns before adding borders or shadows. Let content breathe with generous spacing.

T

Clean Typography

Headings carry personality while body text optimizes scanability and contrast. Clear hierarchy guides the eye.

C

Restrained Color

Primary color for emphasis, neutral surfaces for backgrounds. No off-palette colors, maximum clarity.

Visual Style

Bold & Minimal

Every element serves a purpose. Strong contrast between text and background ensures readability. Typography scales expressively from desktop to mobile.

Consistent vertical rhythm across sections and components creates a predictable, scannable interface. Align columns and modules to a grid—avoid ad-hoc offsets.

Purposeful Interaction

Subtle transitions emphasize primary actions. Short, purposeful animations (150–250ms) with stable easing provide feedback without distraction.

Hover, focus-visible, active, disabled, and loading states are explicit and consistent. Every interaction reinforces the minimal aesthetic.

Color Usage Guidelines

Primary (#0C0C09)

Use for CTA emphasis, primary actions, and key interactive elements. This is your strongest visual signal.

Surface (#F4F4F1)

Use for large backgrounds, cards, and alternating sections. Creates subtle visual separation without borders.

Text (#0C0C09)

Keep body copy on this color for maximum legibility. High contrast ensures readability across all contexts.

Semantic Colors

Success, Warning, and Danger provide clear feedback. Use sparingly to maintain visual hierarchy.

Anti-patterns to Avoid

Off-palette Colors

Do not introduce colors outside the defined palette. If you need a new color, it should be added to the system, not used ad-hoc.

Flattened Hierarchy

Do not use the same type size and weight for all text. Clear hierarchy guides users through content efficiently.

Decorative Effects

Do not add decorative effects that reduce readability or accessibility. Every visual element should serve a functional purpose.

Mixed Visual Metaphors

Do not mix unrelated visual metaphors in the same interface. Consistency builds trust and reduces cognitive load.
