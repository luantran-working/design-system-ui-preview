Simple, powerful API

Integrate OpenAI models into your applications with just a few lines of code. Our API is designed for developers who want to build AI-powered products quickly.

  • RESTful API with comprehensive documentation
  • Official SDKs for Python, Node.js, and more
  • Streaming responses for real-time applications
  • Function calling for structured outputs
  • Fine-tuning for custom use cases
  • Enterprise-grade security and compliance
# Install the OpenAI Python library
pip install openai

# Import and configure
import openai

openai.api_key = "your-api-key"

# Create a chat completion
response = openai.ChatCompletion.create(
  model="gpt-4",
  messages=[
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": "Explain quantum computing"}
  ]
)

# Print the response
print(response.choices[0].message.content)
