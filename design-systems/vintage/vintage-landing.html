Getting Started

Installation

Include the Vintage CSS file and required fonts in your HTML:

<link href="vintage-styles.css" rel="stylesheet"> <link href="https://fonts.googleapis.com/css2?family=Silkscreen:wght@400;700&family=JetBrains+Mono:wght@400;700&display=swap" rel="stylesheet">

Design Principles

  • Use Primary (#008080) for CTA emphasis
  • Keep body copy on Text (#000000) for legibility
  • Maintain consistent vertical rhythm with spacing scale
  • Ensure clear hierarchy: headline → support → action
  • Use whitespace before adding borders or shadows

Typography Guidelines

  • Headings use Silkscreen for personality
  • Body text optimized for scanability
  • Scale: 12/14/16/20/24/32px
  • Weights: 400 (normal), 700 (bold)
  • Monospace code uses JetBrains Mono

Anti-patterns to Avoid

  • Don't introduce off-palette colors
  • Don't flatten hierarchy with same type sizes
  • Don't add decorative effects that reduce readability
  • Don't mix unrelated visual metaphors
  • Don't skip focus-visible states
