Built for developers

Stripe's APIs are designed to be intuitive, well-documented, and easy to integrate. Get started in minutes with our libraries and SDKs.

  • RESTful API with predictable resource-oriented URLs
  • Official libraries in 8+ programming languages
  • Comprehensive documentation and guides
  • Webhooks for real-time event notifications
  • Test mode for safe development
// Create a payment intent
const stripe = require('stripe')('sk_test_...');

const paymentIntent = await stripe.paymentIntents.create({
  amount: 2000,
  currency: 'usd',
  payment_method_types: ['card'],
  metadata: {
    order_id: '12345'
  }
});

// Confirm the payment
const result = await stripe.paymentIntents.confirm(
  paymentIntent.id,
  { payment_method: 'pm_card_visa' }
);

console.log(result.status); // 'succeeded'
