DESIGN PRINCIPLES

Core guidelines for bold, impactful design

✓ DO THIS

  • ✓ Use Primary (#0077BC) for CTA emphasis
  • ✓ Keep hierarchy obvious and scannable
  • ✓ Maintain consistent spacing scale
  • ✓ Use whitespace before borders
  • ✓ Make focus states explicit
  • ✓ Keep microcopy action-oriented

✗ AVOID THIS

  • ✗ Don't use off-palette colors
  • ✗ Don't flatten type hierarchy
  • ✗ Don't add decorative effects
  • ✗ Don't use ad-hoc spacing
  • ✗ Don't mix visual metaphors
  • ✗ Don't use generic filler copy

TYPOGRAPHY SCALE

HEADING 1 - DISPLAY

HEADING 2 - MAJOR SECTION

HEADING 3 - SUBSECTION

HEADING 4 - COMPONENT TITLE

Body text uses regular weight for optimal readability. The Bold design system prioritizes scanability and contrast while maintaining strong visual personality through heavyweight headings.

Secondary text provides supporting information with reduced visual weight but maintains clarity and legibility.
