Symbols & Styles

Prototyping

Version Control

Track changes, restore previous versions, and collaborate without fear. Your design history is always preserved.

Real-Time Collaboration

Work together in the same document simultaneously. See cursors, edits, and comments as they happen.

  • Live multiplayer editing with presence indicators
  • Inline comments and feedback threads
  • Smart conflict resolution
  • Activity history and notifications

Developer Handoff

Bridge the gap between design and development with automatic specs, assets, and code snippets.

  • Automatic CSS, iOS, and Android code generation
  • Export assets in multiple formats and resolutions
  • Inspect mode for precise measurements
  • Design tokens and style guide export
💻

Cloud Libraries

Share design systems, components, and assets across your entire organization with cloud-synced libraries.

  • Centralized component libraries
  • Automatic updates across all documents
  • Team-wide style guides and brand assets
  • Permission controls and access management
