Design Principles

Guidelines that shape every component and interaction

🎯

Clear Hierarchy

Headline → support text → primary action. Use whitespace to separate concerns before adding borders or shadows.

🔍

Readability First

Body text on #141414 for legibility. Never sacrifice readability for decorative effects.

🎨

Consistent Palette

Use existing tokens before introducing new colors. Primary (#1856FF) for CTA emphasis.

⚡

Purposeful Motion

Short transitions (150-250ms) with stable easing. Motion emphasizes interaction, not decoration.

📏

Predictable Grid

Align to consistent grid. Avoid ad-hoc offsets. Maintain vertical rhythm across sections.

🚫

Avoid Anti-patterns

Don't flatten hierarchy with same type size. Don't mix unrelated visual metaphors.
