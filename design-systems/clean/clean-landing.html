Core Principles

Clean design system emphasizes clarity, organization, and usability

W

Ample Whitespace

Generous spacing creates breathing room, making content easier to scan and understand. Built on an 8pt baseline grid for consistent vertical rhythm.

T

Legible Typography

Clear type hierarchy using Poppins for headings and Roboto for body text. Optimized for readability with proper contrast and line height.

C

Limited Color Palette

Focused color system with primary, secondary, and semantic colors. Reduces visual clutter while maintaining clear communication.

A

Accessible by Default

Strong focus-visible states, clear labels, and predictable interactions. Built with WCAG guidelines in mind for inclusive design.

M

Minimal & Modern

Clean aesthetic that prioritizes content over decoration. Subtle transitions and purposeful interactions enhance usability.

O

Organized Layouts

Predictable grid system with consistent internal padding. Clear hierarchy guides users through content naturally.

Typography Scale

Clear hierarchy with Poppins for headings and Roboto for body text

Heading 1 - 48px Poppins Bold

Used for hero sections and primary page titles

Heading 2 - 32px Poppins Bold

Used for section headings and major divisions

Heading 3 - 24px Poppins Semibold

Used for subsections and card titles

Heading 4 - 20px Poppins Semibold

Used for component titles and smaller headings

Body Text - 16px Roboto Regular

Optimized for readability with 1.7 line height. Body text should maintain high contrast against backgrounds for maximum legibility.

Small Text - 14px Roboto Regular

Used for captions, labels, and supporting information
